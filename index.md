# Natzki Group Workspace Index

You can use several shortcuts in the interactive version, such as

- `Shift click` - Open in sidebar
- `t d` - Toggle document (reader) mode
- `t w` - Toggle wide mode
- `g g` - Go to Graph view
- `g h` - Go to Home
- `t t` - Toggle theme.
